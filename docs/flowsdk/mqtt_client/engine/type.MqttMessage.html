Expand description

Alias for MqttPublish (v5) to provide a single, unified type for received messages.

The engine normalizes all incoming PUBLISH packets (whether MQTT v3.1.1 or v5.0) into this structure. This simplifies downstream consumption by providing a consistent API regardless of the protocol version used. For MQTT v3.1.1 messages, the v5-specific fields (Properties) will be empty.
